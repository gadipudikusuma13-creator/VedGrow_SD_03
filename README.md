Library Management System

Description

A console-based Library Management System developed in Java to manage books and library operations efficiently.

Features

- Add Books
- Search Books
- Update Books
- Delete Books
- Issue Books
- Return Books
- Track Book Availability
- Generate Available Books Report

Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner
- File Handling

Project Files

- LibraryManagementSystem.java
- README.md

How to Run

Compile the program:

javac LibraryManagementSystem.java

Run the program:

java LibraryManagementSystem

Sample Operations

- Add a new book
- Search for a book by ID
- Issue a book to a member
- Return a book
- View all available books

Learning Outcomes

- Java Programming
- File Handling
- Collections Framework
- OOP Concepts
- Menu-Driven Applications
